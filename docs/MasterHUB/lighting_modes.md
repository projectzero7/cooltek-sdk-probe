# Lighting Modes

The following lighting effects are all available on supported modules.

| Value | Name | Supports Custom Colors |
| --- | --- | --- |
| `0x00` (`0`) | Disable | No |
| `0x01` (`1`) | Static | Yes |
| `0x02` (`2`) | Breathing | Yes |
| `0x03` (`3`) | Strobe | Yes |
| `0x04` (`4`) | Spring | Yes |
| `0x05` (`5`) | Cycling | No |
| `0x06` (`6`) | Random | No |
| `0x07` (`7`) | Wave | No |
| `0x08` (`8`) | Water | No |
| `0x09` (`9`) | Rainbow | No |
